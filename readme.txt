== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/mentor-icon-pack-for-beaver-builder` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

= Does "Beaver Builder" is Required to use this plugin? =

Yes. You need to install & activate the Beaver Builder plugin first to get access to this addon.

= I used the free version. After purchasing this can I remove it free version? =

Anytime you can delete the free version after the pro version is active and we highly recommended it. Everything works properly.

= I have no Codecanyon account. How can I get the pro version? =

Sorry. At this time, We offer only Codecanyon based service. Please creating an account is very easy.
